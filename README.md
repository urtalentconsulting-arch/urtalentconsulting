# U R Talent Consulting

A clean, professional career guidance website for **U R Talent Consulting**—helping students and job seekers with interview prep, resume building, job readiness, communication skills, assignments support, and more.

## Tech Stack

- **React 18** + **Vite**
- **Tailwind CSS**
- **React Router**
- **Framer Motion**
- **Lucide React** (icons)

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home – Hero, CTAs (Login, Sign Up, Share Your CV), Services, Benefits |
| `/login` | Login (Email, Password) – UI only |
| `/signup` | Sign up (Name, Email, Phone, Password) – UI only |
| `/interview-preparation` | Interview prep, roadmap, mistakes, practice schedule |
| `/job-readiness` | Company expectations, fresher vs experienced, IT vs non-IT |
| `/resume-preparation` | ATS-friendly tips, formats, dos/don'ts, review process |
| `/communication-skills` | English, interview comms, email, body language |
| `/assignments-support` | Assignments, projects, viva, final year project |
| `/classes` | Online/offline, batch, 1:1, who should join |
| `/study-plan` | Daily/weekly plans, skill paths, progress tracking |
| `/contact` | Phone, email, website, form, “Share Your CV” CTA |

## Design

- **Light, clean, professional** – student-friendly and educational
- White / light gray backgrounds, simple colors, modern layout
- Accent: teal (`#0d9488`)
- Navbar with **Login**, **Sign Up**, and **Share Your CV**
- Footer with contact details and quick links
- Responsive, mobile-first layout
